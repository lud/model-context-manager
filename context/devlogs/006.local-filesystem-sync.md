# Local filesystem sync

Phase 4 of the sync feature plan (spec 002).

Added the sync engine for local filesystem sources and the `mcm sync` command.

## What changed

- New `src/lib/sync.ts` — `syncLocalSpec()` implementing all 8 sync rules (merge/mirror modes, file/dir sources, missing sources, type conflicts). File operations logged via `cli.success`/`cli.warning`.
- New `src/lib/sync.test.ts` — 11 tests with real files in temp directories covering all rules plus nested dirs, empty source, and parent dir creation.
- New `src/commands/sync.ts` — interactive command using `@clack/prompts`, iterates sync specs, calls `syncLocalSpec` for localfs, placeholder warning for GitHub.
- New `src/commands/sync.test.ts` — 6 tests mocking project/sync/prompts.
- Registered `syncCommand` in `src/main.ts`.
- New test fixtures in `test/fixtures/sync/`.
